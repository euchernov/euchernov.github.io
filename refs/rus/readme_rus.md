### Ускорение синтеза видов 3d сцен
Полное название магистерской диссертации "Синтез новых видов открытых сцен по реальным фотографиям с помощью NeRF".

С работой можно ознакомиться по ссылке [https://drive.google.com/file/d/1tEAdmg3N4S-FZ_d2hHtUzqb5iekHrg4g/view?usp=sharing]. 

#### Контекст
Синтез новых видов сцены из разреженного набора изображений является давней проблемой компьютерного зрения и необходимым условием для многих приложений дополненной реальности (AR) и виртуальной реальности (VR). Популярное на тот момент семейство методов NeRF [https://arxiv.org/abs/2003.08934] давало хорошие результаты. В частности метод NeRF in the Wild (NeRF-W [https://arxiv.org/abs/2008.02268]) позволял работать с шумными данными (реальными фотографиями открытых сцен), но обучение такой модели могло занимать сутки и более для одной сцены.

#### Идея
Попробуем ускорить обучение NeRF-W [https://arxiv.org/abs/2008.02268] с помощью нового на тот момент метода хеш кодирования на наскольких разрешениях (Multiresolution Hash Encoding) InstantNGP [https://arxiv.org/abs/2201.05989].

#### Что было сделано
Был реализован метод совмещающий NeRF-W и InstantNGP. Были разработаны собственные бенчмарки для тестирования подхода. 

#### Результаты
Разработан подход ускоряющий обучение NeRF-W в 100 раз (с 24 часов до 15 минут) с более высоким качеством по формальным метрикам.

Данный подход был использован на моей первой работе в Sber [https://en.wikipedia.org/wiki/Sberbank] для создание 3d моделей городских достопримечательностей Москвы на основании фотографий из открытых источников.

#### Технологии
PyTorch, tiny-cuda-nn, NeRF, NeRF-W, InstantNGP.
