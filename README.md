# euchernov.github.io

Personal website of **Evgeny Chernov** — ML / AI Engineer.

A place where I share my experience working in machine learning and artificial
intelligence: computer vision research, high-load production systems, and the projects
I've worked on along the way.

🌐 Live site: **https://euchernov.github.io**

## Tech
A static site served by GitHub Pages — plain `index.html`, `styles.css`, and `main.js`
(light/dark theme toggle). No build step.

## Content source
The site content is maintained in English. The source material is kept in Russian under
[`refs/rus`](refs/rus) and must stay in sync with the website (see [RULES.md](RULES.md)).
